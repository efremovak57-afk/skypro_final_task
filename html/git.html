Git

  • git init - команда для инициализации нового репозитория Git в текущей папке.
  • git clone [url] - команда для клонирования существующего репозитория по URL.
  • git status - показывает текущий статус изменений в рабочей директории и индексе.
  • git add [файл или папка] - добавляет изменения в указанный файл или папку в индекс для будущего коммита.
  • git commit -m "сообщение" - сохраняет зафиксированные изменения с описанием.
  • git push - команда для отправки локальных изменений в удалённый репозиторий.
  • git pull - команда для получения последних изменений из удалённого репозитория и их слияния с текущей веткой.
  • git branch - отображает список веток или создает новую ветку.
  • git checkout [ветка или коммит] - переключает текущую ветку или возвращает к предыдущему состоянию.
  • git merge [ветка] - сливает указанную ветку с текущей.
